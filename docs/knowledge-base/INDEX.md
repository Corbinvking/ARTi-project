**Platform URL**: https://app.artistinfluence.com

## How to Use This Knowledge Base

1. **Find the right file** — Use the table above to identify which file covers the topic.
2. **Search by heading** — Each file uses consistent headings: Overview, Features, How To, FAQ, Database Tables.
3. **Check troubleshooting** — For error-related questions, start with `troubleshooting.md`.
4. **Check glossary** — For terminology questions, check `glossary.md`.

## Maintenance

When platform features change, update the relevant knowledge base file. Each file should reflect the current production state of the platform.
